codemelted_web

codemelted_web library

The implementation of the codemelted_web module project. It exports the four core modules that implement the brains of the codemelted module design.

Classes

CAboutData
Data object to capture information about the platform your pwa is running. Supports the CodeMeltedCore.about property.
CActionItem
Data items supporting the construction of different action event based widgets providing information necessary to create the underlying widget.
CAppController
Provides the control of the CAppView state for switching widget content, displaying dialogs, and changing the overall look of the application.
CAppView
This represents the main view for an entire Progressive Web Application. This in combination with the CAppController provide all the tools necessary to quickly construct and control the PWA as a Single Page Application.
CHeaderTitleItem
Provides the title for the header of the CAppView widget.
CLogRecord
Support object for the CLogHandler to allow for additional processing after the initial logging occurs by the module.
CodeMeltedCore
Wrapper API for the codemelted_core library.
CodeMeltedMath
Wrapper API for the codemelted_math library.
CodeMeltedProtocol
Wrapper API for the codemelted_protocol library.
CodeMeltedUI
Wrapper API for the codemelted_ui library.
CProtocol
Base class for a constructed protocol.
CProtocolEvent
Supports the CProtocolHandler object to report data, status, and errors with a constructed protocol via CodeMeltedProtocol API.
CTab
CTabbedView
CWebView
Provides a web view (a.k.a iframe) for the PWA.
